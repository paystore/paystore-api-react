# PayStore - **App Demonstração de Pagamento em React **

## Informações sobre o uso:

O aplicativo tem como finalidade demonstrar de forma simples como deve ser utilizado o mecanismo de pagamento em um projeto react-native.

## Compatível com Payments v1.2.20.0

### Para fazer uso deste app, é necessário estar credenciado a plataforma da Paystore e possuir os demais apps de integração, dúvidas sobre o procedimento seguir a documentação do portal.
